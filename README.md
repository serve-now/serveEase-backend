# Serve-Now 

매장 운영자가 **테이블 → 주문 → 결제 → 매출 리포트**까지 한 번에 관리할 수 있도록 돕는 프로그램 입니다.<br>
점주 대시보드, 매장 직원 단말, 고객 스캐너 등 다양한 채널에서 일관된 API를 제공합니다.<br>
